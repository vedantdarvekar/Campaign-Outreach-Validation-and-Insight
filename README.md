## Problem Statement
Organizations invest heavily in outbound calling campaigns, yet:
- A large portion of calls remain **unconnected**
- Very few calls convert into **completed applications**
- Agent workload and campaign distribution are often uneven

This project answers:
- Where are calls failing?
- Which campaigns and agents perform best?
- What time windows maximize engagement?
- How can conversion rates be improved?

## Key Insights
- **~72% of calls were not connected**, indicating poor contact reach or timing
- Only **2% of total calls resulted in completed applications**
- Campaigns with high volume still showed **low successful outcomes**
- **India contributed the highest call volume**, followed by smaller clusters in Africa and Europe
- Peak engagement occurred between **12:00 PM â€“ 2:00 PM**
- Agent workload was **highly imbalanced**, with a few agents handling most calls

(Insights validated using SQL queries against the raw dataset.) :contentReference[oaicite:3]{index=3}

## Recommendations
- Improve contact list quality to reduce unconnected calls
- Introduce structured **post-call follow-ups** (SMS/WhatsApp/email)
- Optimize calling schedules around **high-conversion time windows**
- Balance agent workload using performance data
- Standardize campaign-to-agent allocation
- Enforce mandatory data fields (e.g., country) during intake
